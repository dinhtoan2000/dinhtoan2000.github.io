Mô tả về khóa học

  • Đây là một khóa học thú vị, thực tế dành cho mọi người
  • Các dự án trong khóa học này được thiết kế để giúp bạn xây dựng mọi thứ bằng cách sử dụng HTML5, CSS & JavaScript mà cần sử dụng libraries hay frameworks.
  • Đây là những dự án nhỏ được thiết kế để bạn hoàn thành trong vài giờ.
  • Trong khóa học này học viên học cách làm việc, thao tác DOM, phát triển thêm những tính năng mới cho dự án của mình.

Một số điều học viên sẽ học được trong khóa học này

  • Tạo layout và UI với HTML/CSS (không sử dụng CSS Frameworks)
  • Javascript cơ bản
  • Javascript ES6
  • Thao tác với DOM
  • HTML5 Canvas
  • Audio & Video API
  • Làm việc với Local Storage
  • High Order Array Methods - forEach, map, filter, reduce, sort
  • setTimout, setInterval
  • Arrow Functions

Khóa học này dành cho ai?

  • Những người muốn cải thiện kỹ năng JavaScript của mình nên tham gia khóa học này.
  • Những người mới bắt đầu lập trình Front End, thông qua khóa học để tự tạo các dự án riêng cho mình.
  • Học sinh cấp 3, học sinh du học.
